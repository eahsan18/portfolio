Loops

A loop is a set of code that is followed while or until a certain condition is met. When that condition is met, the program moves on to the next line of code but until the condition is met, it keeps repeating the code within the loop.

ex: for i in range(3):

t.forward(50)

In real life, this code would be interpreted as "Move 't' forward 50 units 3 times."

Conditionals

Conditionals use boolean to perform certain actions only when certain conditions are true. Conditionals include if, if else, and elif. If a condition is true, it runs through a specific block of code. If it is false, it will run through a different block of code.

ex: enter = input("Do you want to enter?")

if enter == "yes":

print("You have entered!")

else:

print("Error. Wrong Answer.")

In real life, this code would be said as "If the user says yes to entering the building, enter the building. If they do not say yes, display an error message"

Functions

Functions are a section of code that perform a certain task. Many functions are already set up in the computer and allow us to perform small tasks as many times as we want without repeating the code

ex: items = [5,6,7,8]

print(sum(items))

26

In real life, this code would be said as "The sum of the numbers in items, 5, 6, 7, and 8, is 26." The function used here is sum().
