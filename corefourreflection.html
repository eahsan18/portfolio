Variables

Variables are a way to store information onto our computer. They label data with names that are simple for the user and the computer to understand. Variables can be equatede to integers, decimals, words, etc.

ex: herName = "Eshana"

In real life, this code would be said as "Her name is Eshana."

Loops

A loop is a set of code that is followed while or until a certain condition is met. When that condition is met, the program moves on to the next line of code but until the condition is met, it keeps repeating the code within the loop.

ex: for i in range(3):

t.forward(50)

In real life, this code would be interpreted as "Move 't' forward 50 units 3 times."
